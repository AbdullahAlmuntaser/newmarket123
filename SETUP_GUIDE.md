# 🚀 SystemMarket ERP - دليل الإعداد والنشر

## المتطلبات الأساسية

### 1. Flutter SDK
```bash
# تحميل Flutter من https://flutter.dev
flutter doctor  # التحقق من التثبيت
```

### 2. قاعدة البيانات
النظام يستخدم Drift (SQLite) - لا يحتاج إعداد إضافي.

### 3. الاعتماديات
```bash
flutter pub get
```

## 📦 التثبيت

### خطوة 1: استنساخ المشروع
```bash
git clone <repository-url>
cd systemmarket
```

### خطوة 2: تثبيت الحزم
```bash
flutter pub get
```

### خطوة 3: توليد كود Drift
```bash
dart run build_runner build --delete-conflicting-outputs
```

### خطوة 4: تشغيل التطبيق
```bash
flutter run
```

## ⚙️ الإعداد الأولي

### 1. إعدادات النظام
بعد تشغيل التطبيق لأول مرة:
1. انتقل إلى **الإعدادات** → **إعدادات النظام**
2. قم بتكوين:
   - نسبة الضريبة الافتراضية
   - الفرع الافتراضي
   - المستودع الافتراضي
   - إعدادات المخزون

### 2. إنشاء الحسابات المحاسبية
النظام ينشئ تلقائياً شجرة حسابات افتراضية عند أول تشغيل.

### 3. إضافة المستخدمين والصلاحيات
1. انتقل إلى **الإعدادات** → **إدارة الصلاحيات**
2. أضف المستخدمين الجدد
3. حدد الأدوار المناسبة

## 📊 النسخ الاحتياطي

### تصدير البيانات
1. انتقل إلى **الإعدادات** → **النسخ الاحتياطي**
2. اضغط على **إنشاء نسخة احتياطية**
3. سيتم حفظ الملف في مجلد التنزيلات

### استعادة البيانات
1. انتقل إلى **الإعدادات** → **النسخ الاحتياطي**
2. اضغط على **استعادة نسخة**
3. اختر ملف النسخة الاحتياطية
