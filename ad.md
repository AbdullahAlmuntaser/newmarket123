🚨 فحص حقيقي للواجهات والشاشات

تحقق من:

✅ هل الشاشة حقيقية أم وهمية

اكتشف:

- Placeholder pages
- TODO pages
- Mock UI
- Dummy Data
- Fake statistics
- Static reports
- Screens not connected to backend
- Empty buttons
- Dead routes
- Fake actions

إذا وجدت شاشة وهمية:

- قم بتحويلها إلى شاشة تشغيلية حقيقية
- اربطها فعلياً بالـ Database

- نفذ العمليات كاملة

🚨 فحص الربط الحقيقي مع Backend

- هل الحفظ يعمل فعلياً

- هل التعديل حقيقي
- هل البيانات تأتي من Drift/SQLite فعلياً

- هل القيود المحاسبية تُنشأ فعلياً
- هل المخزون يتحدث فعلياً
- هل التقارير تعتمد على بيانات حقيقية

إذا اكتشفت:

- UI غير مربوط
- زر لا يعمل
- عملية ناقصة
- إعداد غير مؤثر
- تقرير وهمي

فقم بتنفيذ الربط الحقيقي الكامل.
