### 2. ✅ خدمة النسخ الاحتياطي (Backup Service)
**الموقع:** `/workspace/lib/core/services/backup/backup_service.dart`

#### الميزات المُنفذة:
- ✅ إنشاء نسخة احتياطية من قاعدة البيانات
- ✅ استعادة نسخة احتياطية سابقة
- ✅ قائمة بجميع النسخ الاحتياطية مع البيانات الوصفية
- ✅ حذف نسخ احتياطية محددة
- ✅ تنظيف تلقائي للنسخ القديمة (أقدم من 30 يوم)
- ✅ حفظ تلقائي للنسخة الحالية قبل الاستعادة
- ✅ بيانات وصفية كاملة (التاريخ، الحجم، الإصدار)

#### كيفية الاستخدام:
```dart
final backupService = BackupService(database);

// إنشاء نسخة احتياطية
final result = await backupService.createBackup(backupName: 'daily_backup');

// استعادة نسخة
await backupService.restoreBackup('/path/to/backup.db');

// قائمة النسخ
final backups = await backupService.listBackups();

// تنظيف النسخ القديمة
await backupService.cleanOldBackups(daysToKeep: 30);
```

### 3. ✅ خدمة التقارير المالية والضريبية (Financial Reports Service)
**الموقع:** `/workspace/lib/core/services/reports/financial_reports_service.dart`

#### التقارير المُنفذة:

##### أ. تقرير ضريبة القيمة المضافة (VAT Report)
- إجمالي المبيعات بدون ضريبة
- إجمالي الضريبة المحصلة
- إجمالي المشتريات بدون ضريبة
- إجمالي الضريبة المدفوعة
- صافي الضريبة المستحقة
- عدد فواتير المبيعات والمشتريات

##### ب. تقرير المبيعات التفصيلي (Sales Report)
- إجمالي الإيرادات
- إجمالي الخصومات
- إجمالي الضريبة
- صافي المبيعات
- إجمالي الكميات المباعة
- عدد الفواتير
- فلترة حسب العميل أو موظف المبيعات

##### ج. تقرير المشتريات التفصيلي (Purchase Report)
- إجمالي المشتريات
- إجمالي الخصومات
- إجمالي الضريبة
- صافي المشتريات
- فلترة حسب المورد

##### د. تقرير الأرباح والخسائر المبسط (Profit & Loss Report)
- إجمالي الإيرادات
- تكلفة البضاعة المباعة
- مجمل الربح
- المصروفات التشغيلية
- صافي الربح
- هامش الربح الإجمالي والنسبة المئوية

##### هـ. تقرير حركة المخزون (Inventory Movement Report)
- حركات المخزون الداخلة والخارجة
- فلترة حسب المنتج
- تفاصيل الحركات (التاريخ، النوع، الكمية، التكلفة)

#### التصدير:
- ✅ تصدير التقارير إلى CSV

#### كيفية الاستخدام:
```dart
final reportsService = FinancialReportsService(database);

// تقرير VAT
final vatReport = await reportsService.generateVATReport(
  startDate: DateTime(2025, 1, 1),
  endDate: DateTime(2025, 1, 31),
);

// تقرير المبيعات
final salesReport = await reportsService.generateSalesReport(
  startDate: DateTime(2025, 1, 1),
  endDate: DateTime(2025, 1, 31),
  customerId: 'customer_123', // اختياري
);

// تصدير إلى CSV
final csvData = reportsService.exportToCSV(vatReport);
```
